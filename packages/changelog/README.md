# @discordx/changelog

> You can use this library without discordx

changelog generator, written in TypeScript with Node.js

# Usage

> npx @discordx/changelog

## Options

The following options are only available

| Name          | Required | Default          | Description                |
| ------------- | -------- | ---------------- | -------------------------- |
| --root        | false    | "./"             | root folder path           |
| --out         | false    | "./CHANGELOG.md" | changelog file path        |
| --tag         | false    | "v\*"            | tag to match with releases |
| --tag-replace | false    | undefined        | repalcement in tag         |
